Detailed Description

template<typename Graph>
class graph::MSTState< Graph >

  • Class to hold the current state of the required datastructures during a MST computing algorithm.

Current state includes the state of the graph, the MST and the span of the MST.. At the end of a MST computing algorithm, MSTState contains the final state which is the return type for every MST computing algorithm.
