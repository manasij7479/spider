Detailed Description

template<typename Graph>
class graph::SSSPState< Graph >

  • Class to hold the source vertex and current state of the required datastructures during a SSSP algorithm.

Current state includes the state of the graph, the distance array(DistanceArray<Graph>) and parent array(ParentArray<Graph>). At the end of a SSSP algorithm, SSSPState contains the final state which is the return type for every SSSP algprithm.
