Detailed Description

  • Bipartite Layout for drawing a graph - The layout plots vertices on two sides of an imaginary demarcation such that no two vertices on the same side has an edge between them. The vertices are plotted at equal distances. This layout is especially useful in plotting graphs such as Complete Bipartite graph etc.
