Detailed Description

template<typename Graph, typename ColorType>
class graph::ColorState< Graph, ColorType >

  • Class to hold the current state of the required datastructures during any graph coloring algorithm.

It colors "ColorType", the template argument, with a color(integer) and stores it in a map. It also stores the no. of colors used. "ColorType" may be a vertex or an edge.
